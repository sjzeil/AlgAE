CS333 Sorting Algorithms

Quick instructions:

  • Select a desired algorithm from the Algorithm menu.
    • Start by generating an array. Keep it small (5-10 elements) - you'll be examining the sorting algorithm behavior in fine detail.
    • Once the array is generated, choose a sorting algorithm.
  • When execution pauses inside of an algorithm, press the "step" or "play" buttons to resume.
  • Observe how each algorithm behaves on random arrays, reversed arrays, and arrays that you have already sorted.

More detailed help is available from the Help menu.
